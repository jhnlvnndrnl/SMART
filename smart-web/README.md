# S.M.A.R.T (Sustainable Monitoring and Agricultural Resource Technology)

![S.M.A.R.T Project](https://img.shields.io/badge/Status-Active-brightgreen)

## About The Project
**S.M.A.R.T** is a hackathon project focused on revolutionizing smart agriculture. It is designed to help farmers monitor crop health, control environmental conditions using Internet of Things (IoT) devices, and predict crop yields by leveraging AI and modern web technologies.

## Key Features
- **Dashboard UI**: Visualize crop health, soil data, and IoT readings in an intuitive interface.
- **IoT Integration**: Monitor temperature, humidity, and soil conditions in real-time.
- **AI-Powered Disease Detection**: Identify crop issues early to prevent widespread damage.
- **Predictive Modeling**: Estimate crop yield based on historical and collected data.

## Tech Stack
- **Frontend**: React.js, Tailwind CSS
- **Backend & Database**: Supabase
- **Automation**: N8n
- **Hardware/Simulation**: IoT Sensors, Unity (Gamified Simulation)
